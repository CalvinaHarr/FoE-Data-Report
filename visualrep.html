Representation 1

This is a representation of height (cm) versus blood pressure of patients tested. To see the correlation between the two factors.

Program:

VR = readmatrix("VR_1.xlsx");

Height = VR(:,2:3:end);

Blood_Pressure = VR(:,3:3:end);

plot (Height,Blood_Pressure,'x')

Representation 2

This represents the average height of males versus females in cm.

Program:

VR = readmatrix("VR_1.xlsx"); MC = 1; WC = 1; Gender = VR(:,1:3:end); for index = 1:length(Gender) if Gender(index) == 1 Man(MC) = VR(MC,2); MC = MC + 1; else Woman(WC) = VR(WC,2); WC = WC + 1; end end Mavg = sum(Man)/length(Man); Wavg = sum(Woman)/length(Woman); graph = [Mavg,Wavg]; Gender = {'Man'; 'Woman'}; bar(graph) set(gca,'xticklabel', Gender) ylim ([0 180])

Representation 3

This is showing the average blood pressure of men versus women, this helps to correlate weather height and blood pressure have a trend.

Program:

VR = readmatrix("VR_1.xlsx"); MC = 1; WC = 1; Gender = VR(:,1:3:end); for index = 1:length(Gender) if Gender(index) == 1 Man(MC) = VR(MC,3); MC = MC + 1; else Woman(WC) = VR(WC,3); WC = WC + 1; end end Mavg = sum(Man)/length(Man); Wavg = sum(Woman)/length(Woman); graph = [Mavg,Wavg]; Gender = {'Man'; 'Woman'}; bar(graph) set(gca,'xticklabel', Gender) ylim([0 80]) ylabel ('Blood Pressure (mmHg)') title('Average blood pressure of Men vs Women')
